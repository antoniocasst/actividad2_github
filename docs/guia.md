# GUIA

## ¿Qué es Git?
Es un sistema de control de versiones que funciona de forma local en tu ordenador. Permite registrar los cambios realizados en los archivos de un proyecto a lo largo del tiempo.

## ¿Qué es GitHub?
Es una plataforma en la nube que permite alojar repositorios remotos de Git. Facilita la colaboración y el almacenamiento de proyectos de forma externa.

## ¿Qué diferencia repo local vs remoto?
* **Repositorio Local:** Es la copia del proyecto que se encuentra físicamente en tu ordenador.
* **Repositorio Remoto:** Es la versión del proyecto que se aloja en un servidor externo como GitHub.
